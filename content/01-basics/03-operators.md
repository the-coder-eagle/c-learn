# 运算符 — 加减乘除与比较

## 1. 算术运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 减法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法 | `5 / 2` | `2` (整数截断!) |
| `%` | 取余 | `5 % 2` | `1` |

### 整数除法的陷阱

```c
int x = 5 / 2;       // 2，不是 2.5！
float y = 5 / 2;     // 2.0，还是整数除法！
float z = 5.0 / 2;   // 2.5 ✅ 至少一个是浮点数
```

### 取余的妙用

```c
num % 2      // 判断奇偶 (0=偶, 1=奇)
num % 10     // 取个位数
num / 10     // 去掉个位数
```

## 2. 赋值运算符

| 运算符 | 等价于 |
|--------|--------|
| `x += 5` | `x = x + 5` |
| `x -= 5` | `x = x - 5` |
| `x *= 2` | `x = x * 2` |
| `x /= 3` | `x = x / 3` |

## 3. 自增自减

```c
int i = 0;
i++;    // 后置：先用后加
++i;    // 前置：先加后用

int a = 5, b;
b = a++;   // b=5, a=6
b = ++a;   // b=7, a=7
```

## 4. 比较运算符

| 运算符 | 含义 |
|--------|------|
| `==` | 等于 |
| `!=` | 不等于 |
| `>` `<` | 大于 / 小于 |
| `>=` `<=` | 大于等于 / 小于等于 |

```c
int age = 20;
printf("%d\n", age >= 18);  // 1 (真)
printf("%d\n", age == 20);  // 1 (真)
```

### ⚠️ `==` 不是 `=`！

```c
if (x = 10)    // ❌ 赋值！恒为真！
if (x == 10)   // ✅ 比较！
```

## 5. 逻辑运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&&` | 与 (AND) | `x>0 && x<10` |
| `\|\|` | 或 (OR) | `x<0 \|\| x>10` |
| `!` | 非 (NOT) | `!ready` |

```c
int score = 85;
if (score >= 60 && score <= 100)  // 两个条件都满足
    printf("Valid\n");
```

## 6. 优先级速查

`()` → `! ++ --` → `* / %` → `+ -` → `< > <= >=` → `== !=` → `&&` → `||` → `=`

**记不住？加括号！**

## 要点速查

- 整数除法会截断，用 `5.0/2` 得 2.5
- `%` 只能用于整数
- `==` 比较，`=` 赋值——绝不混用
- `&&` 全真才真，`||` 有真就真
