# 指针基础 — 地址与解引用

指针是 C 语言最强大的特性，也是初学者最头疼的概念。记住一句话：**指针就是存储地址的变量。**

## 生活中的类比

```
变量 = 一栋房子（里面有数据）
地址 = 房子的门牌号
指针 = 一张写着门牌号的纸条
```

## 两个关键运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `&` | **取地址**：获取变量的内存地址 | `&x` |
| `*` | **解引用**：通过地址访问变量 | `*p` |

```c
int x = 42;
int *p = &x;   // p 存储 x 的地址

printf("x = %d\n", x);    // 42
printf("&x = %p\n", &x);  // 0x7fff... (内存地址)
printf("p = %p\n", p);    // 同上
printf("*p = %d\n", *p);  // 42 (通过指针访问 x)
```

## 声明指针

```c
int *p;      // p 是指向 int 的指针
float *q;    // q 是指向 float 的指针
char *r;     // r 是指向 char 的指针

// 常见写法（都是等价的）
int* p;      // 强调"int 指针类型"
int *p;      // 强调"p 是指针"
int * p;     // 不推荐
```

## 通过指针修改变量

这是指针最重要的用途——在函数中修改外部变量：

```c
void swap(int *a, int *b) {
    int temp = *a;   // 通过指针读取 a 的值
    *a = *b;         // 通过指针修改 a 的值
    *b = temp;
}

int main() {
    int x = 10, y = 20;
    swap(&x, &y);    // 传入 x 和 y 的地址
    printf("x=%d, y=%d\n", x, y);  // x=20, y=10
    return 0;
}
```

**这就是"传地址"——函数可以修改外部变量了！**

## 指针与数组

数组名本身就是指向第一个元素的指针：

```c
int arr[] = {10, 20, 30, 40, 50};

// 以下三种写法等价：
printf("%d\n", arr[2]);      // 30
printf("%d\n", *(arr + 2));  // 30
printf("%d\n", 2[arr]);      // 30 (有趣但别用)

// 用指针遍历数组
int *p = arr;
for (int i = 0; i < 5; i++) {
    printf("%d ", *(p + i));  // 10 20 30 40 50
}
```

## NULL 指针

```c
int *p = NULL;   // p 不指向任何有效内存

// 使用前检查
if (p != NULL) {
    *p = 42;     // 安全
}
```

## 常见错误

| 错误 | 后果 | 正确 |
|------|------|------|
| `int *p; *p = 5;` | 未初始化指针，写到随机地址 | `int x; int *p = &x; *p = 5;` |
| `int *p = 100;` | 指针指向硬编码地址 | 永远不要这样做 |
| `scanf("%d", p);` 忘记解引用 | 改了指针本身而不是值 | `scanf("%d", &x)` 然后 `*p = x` |

## 图解指针

```c
int x = 42;
int *p = &x;
```

```

              ↑              │
              └──────────────┘
```

## 练习

```c
#include <stdio.h>

int main() {
    int a = 10, b = 20;
    int *p = &a;

    printf("a=%d, *p=%d\n", a, *p);  // 10, 10

    *p = 30;  // 通过 p 修改 a
    printf("a=%d\n", a);             // 30

    p = &b;   // 让 p 指向 b
    printf("*p=%d\n", *p);           // 20

    return 0;
}
```

## 要点速查

- `&x` = 取 x 的地址
- `*p` = 访问 p 指向的值
- `int *p = &x` = p 指向 x
- 指针让函数能修改外部变量
- 数组名就是指向首元素的指针
- 永远初始化指针（至少为 NULL）
