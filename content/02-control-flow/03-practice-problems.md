## 10. 猜数字游戏 ★★★

程序随机生成 1-100 之间的数字，用户猜。每次猜完告诉用户"大了"还是"小了"，直到猜对并显示猜了多少次。

```c
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(0));
    int target = rand() % 100 + 1;  // 1-100 随机数
    int guess, tries = 0;

    printf("Guess a number (1-100):\n");
    do {
        printf("Your guess: ");
        scanf("%d", &guess);
        tries++;
        if (guess > target) printf("Too high!\n");
        else if (guess < target) printf("Too low!\n");
    } while (guess != target);

    printf("Correct! You got it in %d tries.\n", tries);
    return 0;
}
```
